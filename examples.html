Usage examples are ready-to-use pieces of code, useful for developers to understand quickly how our WAMP implementation works.

We choose Python and Javascript cause they’re human readable but powerful, and lot of developers use them to build web, mobile and desktop applications. GreenAddress aims to provide developers with a painless and secure integration with the blockchain. Those two languages should cover much of the needs out there, but if you’re looking for another language, please contact GreenAddress.
