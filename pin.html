pin module¶

http://greenaddressit.com/pin/

pin/change_pin_login(pin, device_ident)¶
Changes the PIN for given device_ident
Check usage on Github: JS Example
Parameters:
  • pin (String) – string with a few characters (max 15 characters)
  • device_ident (String) – uuid
Return bool status:
 

Always True if PIN is changed
pin/disable_pin_login(device_ident)¶
Removes the device from the list of PIN logins if the wallet_id matches.
Returns:success - always true
Return type:Boolean
pin/get_devices()¶
Lists all pin devices in this wallet sorted by device_name.
Check usage on Github: JS Example
Returns:[["device_ident", "device_name"], [...]]
Type:List
pin/get_password(pin, device_ident)¶
If pin and device_ident will match, the API will return the paired password.
PIN for the given device is disabled on our servers after 3 failed logins.
Check usage on Github: JS Example
Parameters:
  • pin (string) – The previously set device PIN
  • device_ident (string) – The device name
Returns:

password
Return type:

String
Raises:

#password if there is no such device or the number of attempts has been exceeded
pin/remove_all_pin_logins()¶
Remove all PIN logins previously set. After calling this method, user will not be able anymore to do PIN login from any device he previously paired.
Check usage on Github: JS Example
Returns:success
Return type:Boolean
pin/remove_pin_login(device_ident)¶
Remove the PIN login for given device_ident
Check usage on Github: JS Example
Parameters:device_ident (String) – uuid
Returns:success
Return type:Boolean
pin/set_pin_login(pin, device_name)¶
Generates a device_ident and password and send password back. This password is used for client side seed and mnemonic AES encryption.
Check usage on Github:JS Example
Parameters:
  • pin (String) – with a few characters (max 15 characters)
  • device_name (String) – human readable string (max 100 characters)
Returns:

password
Return type:

String
