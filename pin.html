pin module¶

http://greenaddressit.com/pin/

pin/change_pin_login(pin, device_ident)¶

Change the PIN for given device_ident

Parameters:
  • pin (String) – string with a few characters (max 8 characters)
  • device_ident (String) – uuid
pin/disable_pin_login(device_ident)¶

Remove the device from the list of PIN logins if the wallet_id matches.

Returns:success - always true
Return type:Boolean
pin/get_devices()¶

List all pin devices in this wallet sorted by device_name.

Returns:[["device_ident", "device_name"], [...]]
pin/get_password(pin, device_ident)¶

Find the device_id and return its password. Ignore wallet_id in the db entry.

Returns:password
Return type:String
Raises:#password if there is no such device or the number of attempts has been exceeded
pin/remove_all_pin_logins()¶

Remove the PIN for given device_ident

Parameters:device_ident (String) – uuid
pin/remove_pin_login(device_ident)¶

Remove the PIN for given device_ident

Parameters:device_ident (String) – uuid
pin/set_pin_login(pin, device_name)¶

Generate a device_ident and password and send password back.

Parameters:
  • pin (String) – with a few characters (max 8 characters)
  • device_name (String) – human readable string (max 20 characters)
Returns:

device_id [uuid]
Return type:

String
