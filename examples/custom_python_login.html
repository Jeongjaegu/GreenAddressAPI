Simple Python watch-only login example¶

Here we demonstrate how you can implement a simple Python application which logs in to GreenAddress wallet using custom login, retrieves a P2SH deposit address, and logs out.

Requirements¶

To implement this example, we use the following libraries, so make sure you install them first:
  • Twisted: pip install twisted
  • pyOpenSSL: pip install pyopenssl
  • service_identity: pip install service-identity
  • Autobahn|Python 0.8.12: pip install autobahn==0.8.12
  • pycoin 0.42: pip install pycoin==0.42
  • Requests 2.3.0: pip install requests==2.3.0

Overview¶

To log in to a GreenAddress wallet using custom login, you need a the username and password to the watch_only() call. To retrieve a P2SH deposit address, you simply call fund().

Using the GreenAddress WAMP server¶

Retrieving a token¶

To connect to GreenAddress server, you need to acquire a token over HTTP which is used for protecting our servers against Denial of Service attacks:

import requests

token = requests.get('https://greenaddress.it/token/').text

Logging in¶

Now after retrieving the token, you can log in using custom login:

from autobahn.wamp1.protocol import WampClientFactory, WampCraClientProtocol
from autobahn.twisted.websocket import connectWS
from twisted.internet import reactor
from pprint import pprint

class GreenAddressClientProtocol(WampCraClientProtocol):
   def onSessionOpen(self):
      d = self.authenticate(authKey = token,
                            authExtra = None,
                            authSecret = token)

      d.addCallbacks(self.onAuthSuccess, self.onAuthError)

   def onClose(self, wasClean, code, reason):
      reactor.stop()

   def onAuthSuccess(self, permissions):
      print "Authentication Success!", permissions
      d = self.call("http://greenaddressit.com/login/watch_only", "custom",
         {"username": username, "password": password}, False)
      d.addErrback(lambda failure: failure.printTraceback())
      d.addCallback(pprint)
      d.addCallback(lambda ret: get_p2sh(self))  # get_p2sh is defined in next step below

   def onAuthError(self, e):
      uri, desc, details = e.value.args
      print "Authentication Error!", uri, desc, details

if __name__ == '__main__':
    factory = WampClientFactory("wss://prodwss.greenaddress.it/ws/inv", debugWamp=True)
    factory.protocol = GreenAddressClientProtocol
    connectWS(factory)
    reactor.run()

P2SH generation¶

The fund() call returns a hex-encoded script, so it needs to be hashed to convert it to a P2SH address:

import binascii
from pycoin.encoding import hash160, hash160_sec_to_bitcoin_address

def print_p2sh(hex_script):
   print hash160_sec_to_bitcoin_address(
      hash160(binascii.unhexlify(hex_script)), address_prefix=b'\5')

def get_p2sh(client):
   d = client.call("http://greenaddressit.com/vault/fund")
   d.addErrback(lambda failure: failure.printTraceback())
   d.addCallback(print_p2sh)
   d.addCallback(client.sendClose)  # logout
