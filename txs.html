txs module¶

http://greenaddressit.com/txs/

txs/create_subaccount(pointer, name, user_public, user_chaincode)¶
txs/get_balance([subaccount][, confs])¶

Returns the sum of unspent outputs destined to user’s wallet.

Parameters:
  • String (subaccount) – (default = ‘all’) subaccount; subaccounts are work in progress. Pass ‘all’ to get user’s balance ignoring subaccounts.
  • Number (confs) – number of required confirmations. NOTE: GreenAddress instant confirmed transactions not handled by this call. See get_balance_by_addr for instant confirmations support.
txs/get_balance_by_addr(address[, confs])¶

Returns the sum of unspent outputs destined to given user’s address.

Parameters:
  • String (address) – base58 encoded address
  • Number (confs) – number of required confirmations. NOTE: GreenAddress instant confirmed transactions will be considered to have this number of confirmations, no matter how large it is. For example, given confs=1000000, this call will return balance of user’s instant confirmation outputs.

txs/get_list(last_txhash, unclaimed)¶

Return list of all user’s transactions since last_txhash.

Parameters:
  • String (last_txhash) – Client’s last known txhash. Only newer transactions are returned, with the exception of unclaimed below.
  • unclaimed – List of unclaimed txhashes known by client. These transactions are always returned.
Returns:

{‘fiat_currency’: String, ‘cur_block’: Number, ‘fiat_value’: Number, ‘list’: [...], ‘unclaimed’: {...}}
